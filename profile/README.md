# GraphosCodex

**Collaborative workbench for testing decipherment hypotheses of undeciphered ancient scripts.**

GraphosCodex is being built to fill a real gap in computational decipherment: a versioned, multimodal environment where researchers can register hypotheses, run cross-modal consistency tests, compare proposals from different scholars, and surface the historiographic record of each script's decipherment attempts.

## Status

Currently in specification and setup phase. Code, data, and first published hypotheses coming soon.

## Scope

The first launch will focus on **Linear A** (Bronze Age Crete) and the **Voynich Manuscript** (15th century). Indus, Rongorongo, and Cypro-Minoan to follow.

## Methodology

- Inscriptions and corpora ingested from open sources (SigLA, Mahadevan, Voynich EVA, etc.)
- Each decipherment hypothesis is structured, attributable, and citable
- Tests run across text, iconography, paleography, and historiography
- Reports are reproducible and exportable

## License & openness

Code: open source. Data: open access. Hypotheses: attributed to their authors, citable with permanent IDs.

---

*GraphosCodex is independent and academically aligned. It complements existing projects like INSCRIBE, SigLA, AI-EPIGRAPHY, and DESCRYPT rather than competing with them.*
